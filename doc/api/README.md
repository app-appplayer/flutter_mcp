## Quick Reference

### Core Classes

- **[FlutterMCP](flutter-mcp-api.md#fluttermcp-main-class)** - Main singleton instance
- **[MCPClientManager](flutter-mcp-api.md#client-manager)** - Manages MCP clients
- **[MCPServerManager](flutter-mcp-api.md#server-manager)** - Manages MCP servers
- **[MCPLlmManager](flutter-mcp-api.md#llm-manager)** - Manages LLM integrations

### Configuration Classes

- **[MCPConfig](flutter-mcp-api.md#mcpconfig)** - Main configuration
- **[MCPClientConfig](flutter-mcp-api.md#mcpclientconfig)** - Client configuration
- **[MCPServerConfig](flutter-mcp-api.md#mcpserverconfig)** - Server configuration
- **[LLMConfig](flutter-mcp-api.md#llmconfig)** - LLM configuration

### Plugin System

- **[MCPPlugin](flutter-mcp-api.md#mcpplugin)** - Base plugin class
- **[MCPToolPlugin](flutter-mcp-api.md#mcptoolplugin)** - Tool provider plugin
- **[MCPResourcePlugin](flutter-mcp-api.md#mcpresourceplugin)** - Resource provider plugin
